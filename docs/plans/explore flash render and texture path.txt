7. KEY PARAMETERS TO CONTROL MUZZLE FLASH VISUALS
For tighter, sparklier muzzle-flash appearance:

Texture (m_hTexture): Swap from hl2_muzzleflash / fas_muzzle_break_test to higher-resolution, multi-frame burst (e.g., fas_muzzleflash_test_b at 2048×1024/18 frames, or fire_particle_6 at 2048×512/70 frames)
Animation Rate (m_flAnimationRate): Lower value = slower, more deliberate frame progression; higher = fast flicker (Modern's trail uses 35.0; Povarehok's flame uses 8.0)
Lifetime (m_fLifetimeMin/Max): Shorter = tighter burst; currently Modern flames are 0.04–0.05s
PositionLock (C_OP_PositionLock): If present, each particle stays anchored after spawn, creating a compact burst; if absent, particles drift away
Spawn Offset (m_OffsetMin/Max, m_bLocalCoords): Keep local coords TRUE and offset minimal (1.0 forward) so flash sits tight on muzzle
m_bBlendFramesSeq0: Ensure TRUE for multi-frame textures to avoid jittery steps
Blend Mode (m_nOutputBlendMode): Additive (PARTICLE_OUTPUT_BLEND_MODE_ADD) for bright flashes
Color Init: Bright, warm tones (white-yellow-orange range)
